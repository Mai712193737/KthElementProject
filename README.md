# KthElementProject
 K-th Element of Two Sorted Arrays — Algorithms Project

Given two sorted arrays of size m and n, find the element at the k-th position of the final merged sorted array.

Implements two approaches:
- Iterative (Non-Recursive) → O(k) time | O(1) space
- Recursive (Binary Search style) → O(log k) time | O(log k) space

Built with Java | Data Structures & Algorithms course
